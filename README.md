## 🔑 はじめに — API Key の取得方法

このアプリを使うには、Google の **API Key** が必要です（無料で取得できます）。

![API Key 取得ガイド](docs/images/api_key_guide.png)

### ステップ 1: Google AI Studio を開く

以下のリンクにアクセスしてください：

**👉 [https://aistudio.google.com/apikey](https://aistudio.google.com/apikey)**

> Googleアカウントでログインが必要です。

### ステップ 2: API Key を作成

1. ページ内の **「APIキーを作成」** ボタンをクリック
2. プロジェクトを選択（既存のプロジェクト、または新しいプロジェクトを作成）
3. API Key が生成されます

### ステップ 3: API Key をコピー

- 表示された `AIza...` で始まる文字列をコピーしてください
- このキーをアプリの **「API KEY」** 欄に貼り付けます

> ⚠️ **注意:** API Key は他人に共有しないでください。あなた専用のキーです。

## 💳 支払い設定（Billing）— API Key を使えるようにする

API Key を取得しただけでは、画像生成 API は利用できません。**Google Cloud の支払い設定（Billing）を有効にする必要があります。**

> 💡 **安心ポイント:** 料金は **「従量課金制」** です。月額固定料金ではなく、**画像を生成した分だけ請求**されます。使わなければ **$0（無料）** です。

### ステップ 1: Google Cloud Console を開く

以下のリンクにアクセスしてください：

**👉 [https://console.cloud.google.com/billing](https://console.cloud.google.com/billing)**

> Google アカウントでログインが必要です（API Key を取得したアカウントと同じアカウントを使用してください）。

### ステップ 2: 請求先アカウントを作成する

1. **「請求先アカウントを管理」** または **「課金」** ページが表示されます
2. まだ請求先アカウントがない場合は **「請求先アカウントを作成」** をクリック
3. 以下の情報を入力します：
   - **国** — 日本を選択
   - **アカウントの種類** — 「個人」を選択
   - **お支払い方法** — クレジットカードまたはデビットカードを登録
4. **「送信して課金を有効にする」** をクリック

> 💡 カードを登録しても、API を使わなければ料金は発生しません。

### ステップ 3: プロジェクトに請求先アカウントをリンクする

1. [Google Cloud Console](https://console.cloud.google.com/) で、API Key を作成したプロジェクトを選択
2. 左メニューから **「お支払い」** を選択
3. 作成した請求先アカウントを **「リンク」** します

> ✅ これで API Key が使えるようになります！

### 💰 使いすぎが心配な場合

Google Cloud では **予算アラート** を設定できます。例えば「月 $5 を超えたら通知」のような設定が可能です。

1. [Cloud Console の予算ページ](https://console.cloud.google.com/billing/budgets) を開く
2. **「予算を作成」** をクリック
3. 金額としきい値（例: 50%, 90%, 100%）を設定
4. 設定した金額に近づくとメールで通知が届きます

## 📖 使い方

### 1. API Key を設定する

![設定画面](docs/images/settings_apikey.png)

画面左側の **「設定」** パネルで：
1. **API KEY** 欄に、取得した API Key を貼り付けます
2. ✅ 「設定済み」 と表示されれば OK です
3. キーはブラウザに保存されるため、次回以降は入力不要です

### 2. 生成設定を選ぶ

同じ設定パネルで：
- **アスペクト比** — 生成する画像の縦横比を選択（デフォルト: 16:9）
- **解像度** — 1K / 2K / 4K から選択（高解像度ほど料金が高くなります）

### 3. プロンプトを入力する

画面右側の **「プロンプト」** 欄に、生成したい画像の説明を入力します。

**入力例：**
```
美しい夕焼けの富士山、雲海の上から撮影した風景写真
```

```
カフェラテのラテアート、ハートの模様、木のテーブルの上、
自然光で撮影された写真風
```

> 💡 **ヒント:** 具体的に詳しく書くほど、イメージ通りの画像が生成されます。

### 4. 参照画像を追加する（オプション）

元にしたい画像がある場合：
1. **「参照画像（オプション）」** エリアに画像をドラッグ＆ドロップ
2. または、クリックしてファイルを選択
3. 対応形式: PNG, JPEG, WebP, HEIC, HEIF

### 5. 画像を生成する

- **「✨ 画像を生成」** ボタンをクリック
- または **⌘ + Enter**（ Mac）/ **Ctrl + Enter**（Windows）で生成

生成には数秒〜十数秒かかります。

### 6. 画像を保存する

- 生成された画像の下にある **「💾 保存」** ボタンをクリック
- PNG ファイルとしてダウンロードされます

## 💰 料金について

このアプリ自体は **無料** ですが、画像生成には Google API の料金がかかります。

> 💡 **料金は「従量課金制」です。** 月額固定料金ではなく、画像を生成した分だけ請求されます。使わなければ $0 です。

| 解像度 | 1枚あたりの料金 |
|---|---|
| 1K | $0.134（約¥20） |
| 2K | $0.134（約¥20） |
| 4K | $0.240（約¥36） |

> 参照画像を使用した場合、追加で $0.0011/枚 かかります。
>
> 最新の料金は [Google AI の公式ページ](https://ai.google.dev/pricing) をご確認ください。
>
> ⚠️ API を利用するには [支払い設定（Billing）](#-支払い設定billing-api-key-を使えるようにする) が必要です。

- API Key は **あなたのブラウザ内（localStorage）にのみ保存**されます
